# 📞 Phone Book System

A simple Python phone book application for managing contacts.

## Features

* Add multiple contacts at once.
* Show all contacts.
* Search contacts (case-insensitive).
* Edit existing phone numbers.
* Delete contacts.
* Count contacts.
* Delete all contacts.
* Validate phone numbers (10 digits only).
* Save contacts automatically to a JSON file.
* Load contacts when the program starts.

## Technologies Used

* Python
* Dictionary
* Loops
* Conditions
* JSON

## How to Run

```bash
python phone_book.py
```

## Example Menu

